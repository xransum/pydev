# pydev

## Automated Local Python Installation

To run the script:
```bash
$ bash <(curl -skL https://raw.githubusercontent.com/xransum/pydev/main/pyinstall.sh)
```

**Args**:
- `[version...]`

**Flags and options**:
- `--help` - show help
- `--list` - list available versions
- `--directory <path>` - set installation directory
